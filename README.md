# AuditNet Frontend

Frontend application for the Smart Contract Vulnerability Detector.

## Overview

This is the frontend component of the AuditNet project, providing a user interface for analyzing and detecting vulnerabilities in Solidity smart contracts.

## Getting Started

To get started with the frontend development:

```bash
npm install
npm start
```

## Features

- Smart contract analysis
- Vulnerability detection visualization
- Real-time feedback on code analysis
